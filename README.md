# Socio-economic trends in Polish regions (2020–2025): analysis based on GUS data
### The Main Goal

The aim of this project is to analyze socio-economic trends in Polish voivodeships between 2020 and 2025 using data from Statistics Poland (GUS), and to identify regional differences and development patterns.

### Research questions
1. How did key socio-economic indicators change across Polish voivodeships between 2020 and 2025?
2. Did regional disparities increase or decrease during this period?
3. Which voivodeships experienced the strongest socio-economic growth?
4. Which indicators best describe the level of regional development?
5. Can Polish voivodeships be grouped according to similar socio-economic development patterns?

### Hypotheses
- H1. Voivodeships with higher GDP per capita tend to have higher average wages. 
- H2. Regions with a higher number of businesses per capita tend to have lower unemployment rates.
- H3. Socio-economic disparities between Polish regions remained significant throughout 2020–2025.
- H4. Polish voivodeships can be grouped into clusters with similar development characteristics.
- H5. The strongest socio-economic performance is concentrated in the most urbanized regions.
  
### Scope of the analysis 
- Spatial scope: Poland
- Unit of analysis: Voivodeships
- Time range: 2020 - 2025
- Data Source: Statistics Poland (GUS – Local Data Bank https://bdl.stat.gov.pl/bdl/dane/podgrup/temat)
- Type of analysis: exploratory, statistical, and comparative analysis of regional development
